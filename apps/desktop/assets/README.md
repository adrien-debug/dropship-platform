# Desktop assets

Placeholder icons for the Electron wrapper.

- `tray-icon.png` (16×16), `tray-icon@2x.png` (32×32), `tray-icon@3x.png` (48×48) — menu-bar template icons. Black silhouette on transparent background; macOS inverts them automatically in dark mode.
- `icon.png` (512×512) — generic app icon. **Replace with a real `icon.icns`** before any distribution build (`npm run dist` expects `icon.icns`; with the current `icon.png` Electron Builder will fall back to a generic icon).

All four files are generated by the bootstrap step and are intentionally simple. Do not commit a real icon back over them — drop a proper artwork file next to this README, then update `package.json#build.mac.icon` if the filename changes.
